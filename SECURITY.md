## Security Policy

- Please do not open public issues for security vulnerabilities.
- Report vulnerabilities privately via GitHub Security Advisories:
  https://github.com/nikuscs/ts-code-scan/security/advisories/new
- Alternatively, you can contact the maintainer via GitHub.

### Supported Versions

The latest released version receives security updates.

### Scope

This project is a Rust CLI with no network services. Issues related to unsafe code, command injection, or supply chain (dependencies/workflows) are in scope.
